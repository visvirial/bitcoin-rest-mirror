High Performance Bitcoin Rest Mirror
====================================

Setup
-----

```bash
$ git clone https://github.com/visvirial/bitcoin-rest-mirror.git
$ cd bitcoin-rest-mirror
$ npm install
```

Configure
---------

Create `config.yaml` in the root directory of the project.

Run
---

### Syncer

```bash
$ npm run syncer <coin>
```

### Server

```bash
$ npm run server <coin>
```
